to attach the source and documentation for a particular library
1) right click on the kuali_project in the package explorer and select "Properties"
2) click on "Java Build Path"
3) click on the + sign to the left of the library of interest on the "Libraries" tab
4) select the appropriate category listed below the library (e.g. "Source attachment:") and click the "Edit..." button
5) click the "Workspace..." button and navigate to the appropriate file in the kuali_dependencies project
6) click "OK" 3 times
